Accepted formats

Each field is forgiving about input format:
  • Dotted decimal: 192.168.1.10
  • Binary: dotted octets (11000000.10101000.00000001.00001010) or 32 contiguous bits (11000000101010000000000100001010)
  • Hex: 0xC0A8010A, C0A8010A, or dotted octets like C0.A8.01.0A
  • Integer: any value from 0 to 4294967295
